Embedded NoSQL database for .NET

An open source MongoDB-like database with zero conﬁguration - mobile ready

Install-Package LiteDB

 Standalone database

LiteDB is serverless database delivered in a single DLL (less than 450kb) fully written in .NET C# managed code (compatible with .NET 4.5 and NETStandard 2.0).

Install via NuGet or just copy DLL to your bin project folder.

 Fast and lightweight

LiteDB is a simple and fast NoSQL database solution. Ideal for:

  • Mobile Apps (Xamarin ready)
  • Desktop/local applications
  • Application file format
  • Web applications
  • One database per account/user data store

 And much more...

  • Portable UWP and Xamarin iOS/Android
  • ACID transaction
  • Single datafile (like SQLite)
  • Recovery data in writing failure (WAL mode)
  • Map your POCO class to BsonDocument
  • Fluent API for custom mapping
  • Cross collections references (DbRef)
  • Store files and stream data (like GridFS in MongoDB)
  • LINQ query support
  • FREE for everyone - including commercial use

Features

SQL Syntax

  • LiteDB supports SQL language for data and structure manipulations. You can insert, update, delete or query your database using a very similar SQL relational language.
  • Can be used LINQ expressions (lambda functions) to create fluent API query in your C# code.
  • All SQL commands are supported in new LiteDB.Studio tools.
